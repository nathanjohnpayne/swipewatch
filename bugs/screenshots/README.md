# Bug Screenshots

Store bug evidence and reproduction screenshots here.

**Naming convention:** `issue-<number>-<description>.png` or `<YYYYMMDD>-<HHMMSS>-<description>.png`
Examples: `issue-86-members-loading.png`, `20260401-154810-members-loading.png`

Reference from issues using the `main` branch URL:
```
https://raw.githubusercontent.com/<owner>/<repo>/main/bugs/screenshots/<filename>
```
